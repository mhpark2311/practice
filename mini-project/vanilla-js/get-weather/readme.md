# Get Weather - 날씨 앱

HTML, CSS, JS 기초 연습용

## 기능

- 도시별 기온, 습도, 날씨, 풍속, 날씨 아이콘 표시
- 도시 검색 (엔터키 지원)
- 마지막으로 검색한 도시를 표시
- 최근 검색어 표시 및 클릭으로 재검색
